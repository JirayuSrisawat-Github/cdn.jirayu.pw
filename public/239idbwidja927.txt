เราจะเริ่มด้วยการแยกส่วนของสมการ (x+1)(x+2)(x+3)(x+4) = (A²+Bx+C)² ด้วยสูตรพจน์เต็ม:

(x+1)(x+2)(x+3)(x+4) = (A²+Bx+C)²

เริ่มจากการคูณกระจายของสมการด้านซ้ายมือ:

(x+1)(x+2)(x+3)(x+4) = (A²+Bx+C)(A²+Bx+C)

ต้องการหาค่า A²+B²+C² จากสมการข้างต้น โดยการเปรียบเทียบพจน์ที่มีกำลังเท่ากันของ x:

1. พจน์ค่าคงที่: 1 จาก (x+1)(x+2)(x+3)(x+4)
2. พจน์เทียบกับ x: 2x จาก (x+1)(x+2)(x+3)(x+4)
3. พจน์เทียบกับ x²: 3x² จาก (x+1)(x+2)(x+3)(x+4)
4. พจน์เทียบกับ x³: 4x³ จาก (x+1)(x+2)(x+3)(x+4)
5. พจน์เทียบกับ x⁴: x⁴ จาก (x+1)(x+2)(x+3)(x+4)

เมื่อรวมพจน์ทั้งหมดเข้าด้วยกัน:

x⁴ + 4x³ + 3x² + 2x + 1 = A⁴ + 2A²(Bx) + 2A²(C) + 2Bx(C) + B²(x²) + 2BCx + C²

เปรียบเทียบพจน์ที่มีกำลังเท่ากันของ x และเปรียบเทียบค่าของพจน์เท่ากัน:

1. พจน์ค่าคงที่: 1 = C²
2. พจน์เทียบกับ x: 2x = 2BCx
   จะได้ว่า BC = 1
3. พจน์เทียบกับ x²: 3x² = B²(x²)
   จะได้ว่า B² = 3
4. พจน์เทียบกับ x³: 4x³ = 2A²(Bx)
   จะได้ว่า A² = 2
5. พจน์เทียบกับ x⁴: x⁴ = A⁴ + 2A²(C)
   จะได้ว่า A⁴ + 2A²(C) = 

1

จากนั้นเราสามารถหาค่า A²+B²+C² ได้โดยใช้ค่าที่เราได้หามา:

A² = 2, B² = 3, C² = 1

ดังนั้น:

A² + B² + C² = 2 + 3 + 1 = 6

ดังนั้นค่า A² + B² + C² เท่ากับ 6.

สำหรับสูตรเดียวกับที่เรียนมาในชั้นมัธยมศึกษาปีที่ 3 และการประยุกต์สูตรนั้นสุดยอดนั้น ผมขออธิบายสูตรที่เรียนมาเพื่อให้คุณเข้าใจเพิ่มเติม:

เราสามารถแยกตัวประกอบในสมการดังกล่าวได้เป็นรูปแบบของผลคูณของตัวประกอบทั้งหมด:

(x+1)(x+2)(x+3)(x+4) = (A²+Bx+C)²

จะเห็นว่าทั้งสองฝั่งมีตัวประกอบทั้งหมดเป็นตัวกำลังสอง เราจึงสามารถแยกตัวประกอบดังกล่าวได้เป็นรูปแบบต่อไปนี้:

(x+1)(x+4)(x+2)(x+3) = (A²+Bx+C)(A²+Bx+C)

เราสามารถประยุกต์ใช้สูตรแยกตัวประกอบ (สูตรเอกสารเลขศาสตร์) ที่เรียนมาในชั้นมัธยมศึกษาปีที่ 3 เพื่อแยกตัวประกอบดังกล่าวออกมาเป็นรูปแบบนี้:

(x+1)(x+4)(x+2)(x+3) = (A²+Bx+C)(A²+Bx+C)

ทำการกำหนดตัวแปรให้เรียบร้อย:

a = x+1, b = x+4, c = x+2, d = x+3

โดยนั่นหมายความว่าเราสามารถแทนค่า (x+1) ด้วย a, (x+4) ด้วย b, (x+2) ด้วย c,

 และ (x+3) ด้วย d:

abcd = (A²+Bx+C)(A²+Bx+C)

เราสามารถคูณทั้งสมการด้วยตัวคูณของตัวแปร a, b, c, และ d ได้:

abcd = (A²+Bx+C)(A²+Bx+C)
abcd = (A²+Ba+Ca+BC)(A²+Bx+C)

เนื่องจาก a = x+1, b = x+4, c = x+2, และ d = x+3, เราสามารถแทนค่ากลับเข้าไปในสมการได้:

(x+1)(x+4)(x+2)(x+3) = (A²+Ba+Ca+BC)(A²+Bx+C)

เปรียบเทียบพจน์ที่มีกำลังเท่ากันของ x:

1. พจน์ค่าคงที่: 1 = C²
2. พจน์เทียบกับ x: 2x = 2BCx
   จะได้ว่า BC = 1
3. พจน์เทียบกับ x²: 3x² = B²(x²)
   จะได้ว่า B² = 3
4. พจน์เทียบกับ x³: 4x³ = 2A²(Bx)
   จะได้ว่า A² = 2
5. พจน์เทียบกับ x⁴: x⁴ = A⁴ + 2A²(C)
   จะได้ว่า A⁴ + 2A²(C) = 1

จากนั้นเราสามารถหาค่า A²+B²+C² ได้โดยใช้ค่าที่เราได้หามา:

A² = 2, B² = 3, C² = 1

ดังนั้น:

A² + B² + C² = 2 + 3 + 1 = 6

ดังนั้นค่า A² + B² + C² เท่ากับ 6.

สูตรแยกตัวประกอบในรูปแบบต่างๆ เช่น สูตรเอกสารเลขศาสตร์ จะช่วยให้เราสามารถแ
